# 이미지 파일 추가 안내

이 폴더에 홈페이지에서 사용할 이미지 파일을 추가하세요.

## 필요한 이미지 파일 목록

### 배경 이미지 (권장 크기: 2000px 너비)
- `hero-bg.jpg` - Hero 섹션 배경 이미지 (교재·교구 중심의 따뜻한 책상 이미지)
- `about-bg.jpg` - About 섹션 배경 이미지 (교구와 교재가 놓인 테이블 이미지)
- `cta-bg.jpg` - CTA 섹션 배경 이미지 (사람보다 교구가 중심인 교육 환경 이미지)

### 카드 이미지 (권장 크기: 800px 너비)
- `feature-1.jpg` - 학회지 안내 카드 이미지 (학술지와 논문)
- `feature-2.jpg` - 학회소식 카드 이미지 (알록달록한 문구류)
- `feature-3.jpg` - 교육센터 카드 이미지 (교육용 카드와 교구)

### 뉴스 이미지 (권장 크기: 800px 너비)
- `news-1.jpg` - 공지사항 느낌의 노트와 펜
- `news-2.jpg` - 회의·기록용 교재
- `news-3.jpg` - 교육용 자료와 문서

## 사용 방법

1. 위 목록에 맞는 이미지 파일을 이 폴더에 추가하세요.
2. 파일명은 정확히 위와 같이 지정해주세요.
3. 이미지가 없으면 기본적으로 빈 공간으로 표시됩니다.

## 이미지 최적화 팁

- 웹용으로 최적화된 JPG 또는 WebP 형식을 사용하세요.
- 파일 크기는 각 이미지당 500KB 이하로 권장합니다.
- 이미지 편집 도구를 사용하여 크기를 조정하고 압축하세요.
